トップ 機能 掲示板

コントロールバー

  • 再生
  • 停止
  • 前の動画
  • 次の動画
  • 検索
    投稿日，再生数，マイリスト数の条件で検索できます
  • 設定
  • ミュート，音量の調整
  • 動画タイトル，経過時間，トータル時間，残り時間の表示

Tips

  • 右上のトータル時間の所をクリックすると残り時間が表示されるようになります．
  • サイドバーの幅が小さくなってタイトルが表示しきれなくなると，スクロールを始めます．

動画画面

  • コントロールバーとプレイリスト画面の間にあるつまみを下にドラッグすると動画画面を開くことが出来ます．
  • つまみをクリックすると動画画面を開いたり閉じたり出来ます．

プレイリスト画面

ニコニコ動画，YouTubeの動画をiTunesのようなプレイリストで一緒に管理することが出来ます．
プレイリストにはグリッドビュー，リストビューの2種類のビューがあります．

プレイリストの機能

  • メインウィンドウから追加
    メインウィンドウのタブでニコニコ動画，YouTubeの動画のページを開き，このコマンドを実行すると現在タブで開いている動画をプレイリストに追加できます．
  • 削除
  • 名前変更
  • 更新
    プレイリスト内の動画の情報を更新します．
  • ドラッグで並び替え

プレイリスト内の動画の機能

  • ダブルクリックでロード
  • 削除
  • 更新
    選択された動画の情報のみ更新します．
  • 元のサイトを開く
  • ドラッグで並び替え，プレイリストに追加
    並び替えはリストビューの時のみできます．

Tips

  • プレイリストとビューの幅はドラッグで調節できます．
  • リストビュー
    • 複数動画を選択でき，それぞれの動画に対して削除，更新，ドラッグでプレイリストに追加できます．
    • サムネイルは小さいですがロールオーバーで拡大されます．
    • 各列のヘッダをクリックすることでその項目でソートできます．
    • 右上にあるこのアイコンをクリックすると表示する列を選択できます．
  • グリッドビュー
    • 表示される情報が少ないですが，ツールチップで詳細な情報が見れます．
  • ライブラリについて
    ライブラリは全ての動画を管理するプレイリストです．従って，削除したり名前を変更することは出来ません．
    他のプレイリストは単にこのライブラリ中の動画を参照しているだけです．プレイリストから動画を削除してもライブラリからは消えません．
    しかし，ライブラリから動画を削除すると他のプレイリスト中にある動画も同時に消されるので注意して下さい．

検索画面

検索結果を1ページ当たり30件表示します．

ステータスバー

各種メッセージを表示します．

  • コンテキストメニューが出てきます．
    • 新しいプレイリスト
    • ライブラリをエクスポート
    • ライブラリをインポート
  • シャッフルオン・オフの切り替えが出来ます．
  • リピート再生の状態が変わります．
    • リピートしません
    • プレイリストをリピートします
    • 動画をリピートします
  • グリッドビュー リストビューにそれぞれ切り替えます．

キーボードショートカット

再生/停止 Ctrl + P
前の動画 Ctrl + ←
次の動画 Ctrl + →
新しいプレイリスト Alt + Shift + N

削除 Delete

ログインについて

YouTubeはログインしなくても検索，再生が行えるので問題ないのですが，ニコニコ動画の場合ログインしないと検索・再生共に出来ません．
従って，このアドオンではニコニコ動画に自動ログインする機能があるのですが，それについていくつか説明します．
  • ログインするタイミング
    ニコニコ動画をロードしようとしたか，もしくは検索をしようとしたときにログインのチェックが行われ，ログインしていなければ自動でログインします．
    また，ログインした後も一定時間毎にクッキーの更新を行ってログイン状態を保とうとします．
  • 2重ログインについて
    • ニコニコ動画では2重ログイン（2カ所からログインすること．デスクトップとノートPCから同時にログインしようとする場合などに起こります．）が禁止されています．
    • 2重ログインでクッキーが無効化された時に動画を再生しようとすると，下のような画面になります．
    • この画面になったら再度ログインを試みます．（リロードして再度ログインのリンクをクリックする必要はありません．）
